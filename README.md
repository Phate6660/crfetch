# crfetch

A neofetch-like program written in crystal.

## Running

- Install Crystal and Shards
- Clone:
```
git clone https://github.com/Phate6660/crfetch
cd crfetch
```
- Build: `shards build crfetch`
- Run: `bin/crfetch`
